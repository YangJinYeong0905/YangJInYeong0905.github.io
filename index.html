산점도 (Scatter Plot)

데이터의 두 변수 간의 관계를 시각화하는 데 유용합니다. 예 : 학생 연구 시간 대 시험 점수

예제 데이터 다운로드


                # 코드 복붙할때 들여쓰기 하지 않으면 오류가 날 수 있습니다. 
                # Google-colab 환경에서 Python 코드 예시

                import matplotlib.pyplot as plt

                # 데이터
                x = [1, 2, 3, 4, 5]
                y = [2, 4, 6, 8, 10]
                
                # 산점도 그리기
                plt.scatter(x, y)
                
                # 레이블, 제목 등 추가
                plt.xlabel('X 축')
                plt.ylabel('Y 축')
                plt.title('Matplotlib 산점도 예시')
                
                # 그래프 표시
                plt.show()

히트맵 (Heatmap)

상관행렬과 같은 행렬 데이터를 시각화하여 변수들 간의 상관성을 확인할 수 있습니다. 예 : 높이, 체중 및 연령 간의 관계

예제 데이터 다운로드


                # 코드 복붙할때 들여쓰기 하지 않으면 오류가 날 수 있습니다. 
                # Google-colab 환경에서 Python 코드 예시

                import matplotlib.pyplot as plt
                import numpy as np
                
                # 히트맵에 표시할 데이터 (3×3 행렬)
                data = np.array([
                    [1,   0.5, 0.3],
                    [0.5, 1,   0.2],
                    [0.3, 0.2, 1]
                ])
                
                # X, Y축 레이블
                x_labels = ['Feature1', 'Feature2', 'Feature3']
                y_labels = ['Feature1', 'Feature2', 'Feature3']
                
                # 그래프 객체(figure)와 축(axes) 생성
                fig, ax = plt.subplots()
                
                # 히트맵 생성
                heatmap = ax.imshow(data, cmap='viridis', interpolation='nearest')
                
                # X축, Y축 위치에 맞춰 레이블 설정
                ax.set_xticks(np.arange(len(x_labels)))
                ax.set_yticks(np.arange(len(y_labels)))
                ax.set_xticklabels(x_labels)
                ax.set_yticklabels(y_labels)
                
                # X축 레이블이 겹치지 않도록 회전 (필요시)
                plt.setp(ax.get_xticklabels(), rotation=45, ha='right')
                
                # 컬러바(색상 범례) 표시
                plt.colorbar(heatmap)
                
                # 그래프 제목 (선택)
                ax.set_title('히트맵 예시')
                
                # 그래프 표시
                plt.show()

Google Colab에서 데이터 분석하기

1. Google Colab 시작하기

Google Colab은 Python 코드를 클라우드 환경에서 실행할 수 있는 강력한 도구입니다. 다음 단계를 따라 시작하세요:

  1. Google Colab에 접속합니다.
  2. Google 계정으로 로그인한 후 새 노트를 만듭니다.
  3. Python 코드를 입력하고 실행할 수 있는 인터페이스를 확인하세요.

2. 공공 데이터 다운로드하기

국내 공공 데이터를 다운로드하여 분석할 수 있습니다:

  1. 공공데이터 포털에 접속합니다.
  2. 검색창에 필요한 데이터를 입력하고 원하는 데이터를 선택합니다.
  3. CSV 형식으로 데이터를 다운로드합니다.
  4. 구글 드라이브에 저장합니다.

실제 활용 예시

1. 광산구 인구 분석

광산구에서 17세부터 19세까지의 인구 분석 결과:

  • 광산구는 광주 내에서 17-19세 인구가 가장 많은 구입니다.
  • 수완동은 광산구 내에서 가장 인구가 많은 동으로 확인되었습니다.

# 데이터 로드 및 분석
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df_population = pd.read_csv('/content/drive/MyDrive/광주광역시_연령별_인구수_2020.csv', encoding='cp949')
gwangsan_population = df_population[(df_population['구'] == '광산구') & (df_population['연령'].between(17, 19))]
population_summary = gwangsan_population.groupby('동').sum()['합계']
print(population_summary)

2. 수완동 내 업종 분석

수완동의 제과점 수가 적은 것을 분석하여 신규 사업 기회를 제안합니다.


# 수완동 업종 분석
df_business = pd.read_csv('/content/drive/MyDrive/광주광역시 광산구_음식점 현황.csv', encoding='cp949')
suwan_business = df_business[df_business['소재지지번주소'].str.contains('수완동')]
bakery_count = suwan_business[suwan_business['업종명'] == '제과점영업'].shape[0]
print(f"수완동 내 제과점 수: {bakery_count}개")

3. 학생 교통 통계

수완동 학생들의 버스 이용 통계를 분석한 결과:

  • 일평균 등교: 126명
  • 일평균 하교: 133명

# 교통 데이터 분석
df_transport = pd.read_csv('/content/drive/MyDrive/광주광역시_교통카드_이용통계.csv', encoding='cp949')
filtered_transport = df_transport[df_transport['BUSSTAT_PATH_NUMBER'].str.contains('수완')]
total_commute_in = filtered_transport['BUSSTAT_GFF_YNGBGS_ATTEND'].sum()
total_commute_out = filtered_transport['BUSSTAT_GFF_YNGBGS_LVFFC'].sum()
print(f"등교 인원: {total_commute_in}, 하교 인원: {total_commute_out}")
